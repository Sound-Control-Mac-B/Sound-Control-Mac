## Overview

Sound Control Professional Suite is a complete per-application audio management platform that serves audio professionals and power users with the routing depth, keyboard control, and automation needed for sophisticated multi-application audio workflows.

Keyboard volume control overrides the system volume keys to control the currently focused application's volume rather than the system volume, enabling quick adjustment of whichever application is in use without opening the Sound Control panel. Modifier key combinations control specific applications regardless of focus.

Audio recording captures any application's audio output as a file, enabling recording of streaming audio, video call audio, or any other application output without a separate recording tool. The recording captures the post-EQ audio, preserving any equalization adjustments.

Super Volume boosts application volume beyond the normal maximum, providing additional headroom for applications with quiet output. The boost applies clean gain without the distortion that system-level volume overdrive produces.

Sound Control Professional Suite is the audio control platform for users who need professional-grade per-application audio management with the routing, recording, and keyboard control features that multi-application audio workflows demand.

## Performance & Integration

Sound Control is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Sound Control processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
